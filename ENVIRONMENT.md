# 電腦環境安裝與設定

## 實作環境說明

為了能讓大家的環境基本上都差不多能在課堂上順利地進行實作，請務必在上課前把需要的軟體全部安裝好，以下是安裝的相關軟體與安裝步驟與說明，如果安裝過程有遇到任何問題，請向**紙鈔**諮詢。

### 作業系統

- Windows 8.1 以上版本 (更新到最新 Service Pack 版本)

### 瀏覽器

- [Google Chrome](http://www.google.com/intl/zh-TW/chrome/)

### 文字編輯器

- Sublime Text 3
